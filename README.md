# Сравниваем вакансии программистов

Программа собирает и анализирует данные о вакансиях для различных языков программирования из двух источников: [Habr](https://career.habr.com) и [SuperJob](https://superjob.ru).

#### Функционал

- Загружает вакансии из Habr и SuperJob
- Высчитывает среднюю зарплату по каждому языку программирования
- Выводит результат в таблице в консоль

> по умолчанию используется город Москва.
---
### Как установить

1. Создайте и активируйте виртуальное окружение:
```
python -m venv .venv
.\.venv\Scripts\activate
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Создайте файл .env в корне проекта и внесите ваш ключ от API SuperJob:
```
SJOB_SECRET_KEY=ваш_ключ_с_superjob
```
> Получить ключ можно на ```https://api.superjob.ru/ ```
---
### Запуск

```
python main.py
```
---
### Результат

Программа выведет в консоль две таблицы:

- Средняя зарплата по языкам на Habr (Москва)
- Средняя зарплата по языкам на SuperJob (Москва)
